# Bond React Boiler Plate

General boilerplate repo for build Single Page Applications in Bond

### Installation

```sh
npm install
```

### Build Scripts

```sh
gulp dev: for local dev enviornment and live reload

gulp build: To run build script

node server: node server.js
```
